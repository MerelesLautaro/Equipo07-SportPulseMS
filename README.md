# Instalación

## 1. Clonar repositorio

```bash
git clone https://github.com/MerelesLautaro/Equipo07-SportPulseMS.git
```

## 2. Ejecutar con Docker
```bash
docker-compose up --build
```

## Health Check
```bash
GET /actuator/health
```
